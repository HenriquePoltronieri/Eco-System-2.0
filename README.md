# 🌱 Eco System 2.0

## 📖 Sobre o projeto
O **Eco System 2.0** é um projeto desenvolvido em **Kotlin**, com foco na simulação e organização de um ecossistema digital, explorando interações entre diferentes entidades de forma estruturada.

O projeto busca aplicar boas práticas de programação, orientação a objetos e organização de código, servindo também como base para estudos e evolução contínua.

---

## 🚀 Tecnologias utilizadas

- 💻 **Kotlin**
- ☕ **JVM (Java Virtual Machine)**
- 🔧 **Gradle** *(ou Maven — ajuste se necessário)*
- 🧰 Git & GitHub
